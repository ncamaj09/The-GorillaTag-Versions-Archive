Welcome to The GorillaTag Version Archive!

This website provides old versions of the VR game GorillaTag versions from 2020 - 2022. This archive was made too experience old versions of the game or just to go back in time. The files provided in all of the versions listed in the versions tab are downloadable apk files with mega links. the apk's will only work for Oculus headsets. in order to download these versions on to your headset you must unable developer mode and then after unabling developer mode download any file explorer that works on the Oculus headset. The Oculus's operating system is android but modified for VR games. Then download any version of the game from this archive and then go to your file explorer and go to your downloads folder. After going to your downloads folder click on the version of the game that you installed from the website and after that it should install the version of the game. If it says that it could not install that version you must delete GorillaTag from your headset and reinstall the downloaded apk also when you delete GorillaTag it will save your data because GorillaTag uses Proton and Playfab ID's. Also this website is on V2 because the old websites links were just weird so I wanted to create the bigger version of the website. Also my insperations from creating this website is OmniArchive and GorillaTag because omniarchive stores old versions of minecraft like a Database and GorillaTag because I started playing the game when it was christmas of 2022 and i really wanted to visit that version again.
