Info about this site

This is an archive full of official Gorilla Tag Versions from 2020 - 2023. This archive was made so the old versions many of the community grew up with are still alive, functional, and work properly.

Important Notice

Now there are only .APK and PCVR zip files. Oculus headsets are basically Android devices but modified for VR gaming. I would suggest getting a file Explorer on your Oculus headset. I would recommend Google Files or CX File Explorer. You would have to get these file explorers by using the SideQuest Advanced Installer to sideload the APK file onto the headset as an unknown app. The link for the SideQuest Advanced Installer will be in the Social Tab.
